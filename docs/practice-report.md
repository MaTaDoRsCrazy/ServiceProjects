# Отчет по практике

## Тема проекта

Corporate ServiceDesk - информационная система учета заявок и оборудования организации.

## Цель работы

Разработать прикладную информационную систему с backend, базой данных, web-интерфейсом,
desktop-приложением, мобильным клиентом и контейнеризацией.

## Выполненные работы

- Проанализирована предметная область ServiceDesk.
- Определены роли пользователей: администратор, техник, сотрудник.
- Спроектирована структура базы данных PostgreSQL.
- Разработан backend на Java Spring Boot.
- Реализована JWT-авторизация.
- Созданы REST API для заявок, оборудования, комментариев и отчетов.
- Разработана web-админка.
- Разработан desktop-клиент на Kotlin/JVM.
- Разработан Android-клиент на Kotlin.
- Подготовлен запуск через Docker Compose.
- Подготовлены JUnit-тесты.
- Оформлена техническая и пользовательская документация.
